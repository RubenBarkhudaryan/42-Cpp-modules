# 42 C++ Modules Journey

This repository tracks my progress through the **42 School C++ Modules** curriculum.

The core idea of this cursus is to build a solid C++ foundation step by step: from syntax and object-oriented basics to advanced topics like polymorphism, templates, STL containers, and algorithms.

## What I Have Done

### Module 00 - C++ Basics and First Classes
- `ex00` Megaphone: command-line input/output and string handling.
- `ex01` PhoneBook: introducing classes, encapsulation, and simple data management.
- `ex02` Account: static members, class-level state, and formatted logs.

### Module 01 - Memory, References, and Practical OOP
- `ex00` to `ex02`: stack vs heap allocation, constructors/destructors, zombie exercises.
- `ex03` Weapon/HumanA/HumanB: references vs pointers in object relationships.
- `ex04` File replace: file streams and text processing.
- `ex05` and `ex06` Harl: member-function pointers and control flow design.

### Module 02 - Orthodox Canonical Form and Operator Overloading
- `ex00` to `ex02`: fixed-point number class, canonical form, and overloaded operators.
- `ex03` BSP: immutable points and geometric logic with custom numeric types.

### Module 03 - Inheritance and Class Hierarchies (Completed)
- Exercises covering single and multiple inheritance, protected members, virtual methods, and proper resource management in derived classes.
- Implementations and tests are available in the `module-03/` directory.

### Module 04 - Subtype Polymorphism and Abstract Interfaces (Completed)
- Exercises focused on polymorphism, abstract base classes, pure virtual functions, and runtime behavior through interfaces.
- Implementations and examples are available in the `module-04/` directory.
